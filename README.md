# Monte-Carlo-Simulation-of-Neutron-Slowing-Down
Monte Carlo simulation of neutron moderation. Reproduces spectral hardening and 1/v tail behavior based on Coveyou et al. (1956) . Validates the Wigner-Wilkins (1944) relationship (T_m/T_e) using C and Python. Features 1,000,000 histories and rejection sampling.
